Пример: a = 0; b = 1; a = a + b;
Для запуска программы требуется:
1. Открыть папку bin в Visual Studio code 
2. В терминале написать ./makeit и убедиться, что на диске D установлен minGW
3. Запустить программу ./test.exe
   
Пример ввода:
a=2;b=3;c=4;d= a+c;
вывод: 
a = 2.000000
b = 3.000000
c = 4.000000
d = 6.000000
